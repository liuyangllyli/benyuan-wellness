# 在 Cursor 里用 Claude Code — 就做这几步

你只需要按顺序做下面这件事。

---

## 第一步：在 Cursor 里打开终端

- 键盘按：**Ctrl + `**（` 是数字 1 左边那个键）
- 或者菜单：**终端 → 新建终端**
- 窗口下面会出现一块可以打字的黑色/蓝色区域，那就是终端。

---

## 第二步：在终端里输入并回车

在终端里输入（可以复制粘贴）：

```
claude
```

然后按 **回车**。

---

## 第三步：按提示在浏览器里登录

- 终端里会提示「Opening browser for authentication...」，并**自动打开浏览器**。
- 在浏览器里用你的 **Claude 账号**登录（没有就注册一个）。
- 登录/授权完成后，浏览器会显示**一串字母数字（code）**。
- **复制这串 code**，回到 Cursor，在终端里**粘贴**，再按 **回车**。

---

## 第四步：开始用

- 终端里出现 Claude 的欢迎语后，就可以在下面**直接打字和 Claude 对话**了。
- 比如你可以说：「帮我看看这个项目里 src 目录的结构」
- 这就是在 Cursor 里用 Claude Code 了。

---

## 如果第二步输入 `claude` 后说「找不到命令」

说明本机还没装好 `claude` 命令，按下面做一次安装：

1. 按 **Win 键**，搜索 **PowerShell**，**右键 → 以管理员身份运行**。
2. 在 PowerShell 窗口里输入下面这一整行（可复制粘贴），回车：

   ```powershell
   irm https://claude.ai/install.ps1 | iex
   ```

3. 等它安装完（会有提示）。
4. **关掉 Cursor，再重新打开** Cursor。
5. 回到 **第一步**，在 Cursor 的终端里再输入 `claude` 并回车，然后按 **第三步** 在浏览器里登录。

---

## 总结（一句话）

**在 Cursor 里按 Ctrl+` 打开终端，输入 `claude` 回车，按提示在浏览器登录并把 code 粘贴回终端，就能用了。**
