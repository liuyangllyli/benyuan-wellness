# 参考原型说明（本元养生 · BenYuan Wellness）

本目录存放**参考原型**的 UI 与业务逻辑说明，供 Cursor 在实现 uni-app 时还原界面与流程。  
**项目名称**：本元养生（中文）/ BenYuan Wellness（英文）。

---

## 内容说明

- **参考原型**：由外部工具生成的 Web 原型（React 技术栈），用于确定「各页长什么样、流程怎么走、数据写哪里」。
- **本仓库实现**：在 Cursor 中用 **uni-app** 重新实现 APP + 微信小程序，不直接使用参考原型的代码仓库。
- **本目录**：不存放参考原型的完整源码（因其技术栈不同），只存放**从参考原型提炼的、与实现相关的说明**。

---

## 文件清单

| 文件 | 说明 |
|------|------|
| [每页功能与逻辑要点.md](./每页功能与逻辑要点.md) | 各页面（首页、测评、炼体、食养、觉察、归元等）的功能、数据读写、入口出口、关键逻辑要点；实现时按此还原即可。 |

---

## 使用方式

- 在 Cursor 中实现某一页时，打开 `docs/reference/每页功能与逻辑要点.md` 对应小节，结合 `docs/key-code-index.md` 的路径与 localStorage 键名，用 uni-app 实现相同功能与数据流。
- 若后续有新的参考截图或交互说明，可追加到本目录（如 `某页交互说明.md`），并在本 README 中列出。
